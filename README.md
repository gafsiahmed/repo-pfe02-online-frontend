
# Repo PFE02/2025 — Frontend Part

**Dépôt contenant le code de chaque séance de la cohorte 02/25 — Préparation PFE en ligne (Partie Frontend)**

## Description du projet

Ce dépôt regroupe l’ensemble du code front-end développé durant les séances de la formation **« Préparation PFE 01/25 »**.

Chaque séance est organisée dans un dossier distinct (`seance1`, `seance2`, etc.), ce qui permet de suivre l’évolution du projet étape par étape.  
Il est conçu pour :
- revoir le code écrit en classe,
- faciliter les révisions,
- servir de support de référence,
- et partager facilement les travaux réalisés pendant les séances.



---

## Comment utiliser / exécuter le code

1. Clonez le dépôt :

```bash
git clone https://github.com/gafsiahmed/repo-pfe02-online-frontend.git
cd repo-pfe01-frontend-part
````

---

## Remarque importante

Pour les projets **React** présents dans les séances **11 ...**, n’oubliez pas d’installer les dépendances avant d’exécuter le projet.

### Exemple :

```bash
cd Seance11-first-app
npm install
```

Ensuite, vous pouvez lancer le projet avec :

```bash
npm start
# ou
npm run dev
```
